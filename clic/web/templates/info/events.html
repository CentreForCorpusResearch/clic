CLiC 1.4 is now available!

by J. de Joode

Posted on May 5, 2016

A new version of CLiC has been released. Here are a number of highlights in the latest release:

  • The Keyword Extraction Algorithm has completely been refactored. Its results now match the log likelihood ratio formula as explained by Rayson and Garside (2000, cf. Documentation). The keywords are now extensively tested and the clusters and keywords pages contain the information of the parameters.
  • The non-quote extraction algorithm has been improved to contain selection before the first quote in a chapter. This affects concordance searches, keywords and clusters.
  • We have renamed the parameters in URLs so one can easily edit them and share the URLs with other researchers.
  • We have fixed the concordance searches in short suspensions.
  • Keywords and clusters for 3- and 4-grams in short suspensions are now available.

We are still working on more fine-grained token and cluster counts. Currently, non-quote and quote token counts do not add up to the total token count.

Based on feedback from our users, we have fixed the column width in the keywords and clusters pages and we have added a number of help pages that explain basic usage. We have also fixed a bug that resulted in 1-grams when 5-grams were asked.

Find out more about our release at our Github code repository.
